# homebrew-shelly

Homebrew tap for [Shelly](https://github.com/shreyasugemuge/shelly) — a modular zsh configuration with dev workspace and system monitoring.

## Install

```sh
brew tap shreyasugemuge/shelly
brew install shelly
```

## Activate

After installing via Homebrew, activate the zsh configuration:

```sh
shelly install
exec zsh
```

## Update

```sh
brew upgrade shelly
shelly update
exec zsh
```

## Uninstall

```sh
shelly uninstall
brew uninstall shelly
brew untap shreyasugemuge/shelly
```
